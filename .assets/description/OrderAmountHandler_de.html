Sie betreiben einen Online-Shop, für den Ihre Kunden Gutscheine erhalten oder Guthaben ansparen können? Dann kennen Sie sicherlich die Herausforderung, Bestellungen mit einem Bestellwert von 0 EUR abzuwickeln. Denn verschiedene Zahlungsanbieter wie PayPal, PayOne oder ähnliche lehnen Buchungen mit einem Wert von 0 EUR ab.

Unser Plugin OrderAmountHandler schafft ab sofort Abhilfe für dieses Problem, sodass Ihre Kunden jederzeit ganz unkompliziert bei Ihnen bestellen können – auch, wenn Sie dank eines Gutscheins oder Guthabens nichts für die Bestellung bezahlen.

Denn das Plugin prüft den Bestellwert inklusive der Versandkosten und wechselt im Hintergrund die Zahlungsart aus, die von Ihrem Kunden gewählt wurde, falls der Wert der Bestellung Null ergibt. So kann die Bestellung trotzdem wie gewohnt durchgeführt werden. Nach Abschluss der Bestellung setzt das Plugin die gewählte Zahlungsart Ihres Kunden wieder zurück.

Das Plugin steht kostenlos und quelloffen zur Verfügung. Über GitHub können sich Entwickler an der Weiterentwicklung des quelloffenen Plugins beteiligen.
