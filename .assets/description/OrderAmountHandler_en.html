Do you operate an online shop for which your customers can receive vouchers or save credits? Then you surely know the challenge to handle orders with an order value of 0 EUR. Because different payment providers such as PayPal, PayOne or similar refuse bookings with a value of 0 EUR.

Our plug-in OrderAmountHandler can now remedy this problem so that your customers can easily order from you at any time - even if you do not pay for the order thanks to a voucher or credit.

Because the plugin checks the order value including the shipping costs and changes in the background the payment method, which was selected by your customer, if the value of the order is zero. So the order can still be carried out as usual. After completion of the order, the plugin resets the selected payment method of your customer.

The plugin is available free of charge and open source. About GitHub developers can participate in the further development of the open source plugin.
